模块化API文档生成系统

通过分离模板与数据，轻松创建和管理多个API文档项目

系统使用指南

  1. 创建API数据文件：在apiData目录下创建JSON文件（如plugin.json）
  2. 定义API结构：按照指定格式编写API数据： 
    {
  "name": "项目名称",
  "description": "项目描述",
  "classes": [
    {
      "name": "类名",
      "icon": "FontAwesome图标名",
      "category": "类别",
      "description": "类描述",
      "methods": [ ... ],
      "properties": [ ... ],
      "events": [ ... ]
    }
  ]
}
  3. 添加项目：在APIDocumentGenerator类的projects数组中添加新项目
  4. 切换项目：通过顶部下拉菜单选择不同API文档
  5. 自定义样式：修改CSS中的颜色变量来自定义主题

提示： 所有API数据都存储在浏览器内存中，实际应用中应通过AJAX加载外部JSON文件
