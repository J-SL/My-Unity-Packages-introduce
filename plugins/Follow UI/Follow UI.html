Follow UI

API

FollowManager:void AddWorldObject(O worldObject)
添加物体进入UI管理器，之后如果UI出现在摄像机范围内，则会被UI管理器指定的UI跟随。
FollowManager:void RemoveWorldObject(O worldObject)
从UI管理器管理里移除物体。
FollowManager:bool useScreenSpcae
使用屏幕空间坐标系。
FollowManager:folat scaledUI
UI的scale受物体与摄像机的位置影响。
FollowManager:folat scaleParam
缩放UI的参数。
FollowManager:IFllowUIInfo prefab
跟随物体的UI预制体。
FollowManager:List allWorldObjectsList
序列化要跟随的物体列表。
IFllowUIInfo:Transform P_Transform
UI根物体的transform。
IFllowUIInfo:Renderer P_RectTransform
UI根物体的RectTransform。
IWorldObjectInfo:Transform P_Transform
需要被跟随的物体的transform。
IWorldObjectInfo:Renderer P_Renderer
需要被跟随的物体的Renderer。
IWorldObjectInfo: void InitUIInfo(UIInfo info);
UI被生成或被指定给某个物体时调用一次，可以缓存UIInfo用于后续操作UI。
